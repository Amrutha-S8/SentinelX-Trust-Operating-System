### Step 1: Create Repository on GitHub Website

1. **Go to GitHub**: https://github.com/new
2. **Repository Name**: `SentinelX-Trust-Operating-System`
3. **Description**: `Complete 10-layer Trust Operating System with real-time security scoring, ML behavioral analysis, approval workflows, and attack simulation`

5. **DO NOT** initialize with README, .gitignore, or license (we already have them)
6. **Click "Create repository"**

### Step 2: Push Your Code

After creating the repository, GitHub will show you commands. Use these:

```bash
# Navigate to your project
cd C:\Users\Amrutha.S\.kiro\SentinelX

# Add the GitHub repository as remote origin
git remote add origin https://github.com/YOUR_USERNAME/SentinelX-Trust-Operating-System.git

# Rename branch to main (GitHub default)
git branch -M main

# Push your code to GitHub
git push -u origin main
```

Replace `YOUR_USERNAME` with your actual GitHub username.

### Step 3: Alternative - Create Repository via Commands

If you prefer to use commands, copy and paste this into PowerShell:

